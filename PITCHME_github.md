## milestone
大枠の予定を建てることが出来ます

デザインであれば、制作の一つのフェーズを抜き出す

- date: 2017/xx/xx

## issues
milestoneに紐付いたissueを作ります。

- assignee : issueの関係者を登録
- label : どんなカテゴリに属したissueなのかを明示
