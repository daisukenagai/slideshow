# gitとgithub
+++
## git
分散型バージョン管理システム
### ↓↓↓
一度編集したファイルを過去の状態に戻したり、編集箇所の差分を表示したりすることができます。
+++
## github
githubはgitのクラウド・ホスティングサービス
### keyword
ソーシャルコーディング
コミュニケーションの機能が充実している

+++
# 何がいいのか？
いろんな管理ツールがあるのに、なぜgithub?

+++
# ソースコードと結びついているから
ソースコード = 成果物  
成果物と管理（進行や品質）を密接に結びついているから、  
状況の把握がしやすい！
